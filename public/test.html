Acme E-Commerce Shop

This is a sandbox host website demonstrating the AI Customer Support Chatbot Widget. Look at the bottom-right corner of the page to find the chat floating button!

Instructions:

  1. Start `npm run dev` in the terminal.
  2. Open `http://localhost:3000` (Platform landing page).
  3. Go to Enter Dashboard (use any email/password to log in in Sandbox mode).
  4. Open this page at `http://localhost:3000/test.html` in another tab.
  5. Open the Chatbot bubble, fill in the pre-chat form (Sarah Connor) and click Start Conversation.
  6. Message the chatbot. Open the Live Inbox tab in the dashboard side-by-side to watch the real-time agent takeover and manual override in action!
